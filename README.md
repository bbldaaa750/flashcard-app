# Flashcard App

Это полноценное веб-приложение для обучения с помощью флеш-карточек, созданное в рамках курсовой работы по дисциплине "Основы программирования".

## Основные возможности

*   **Аутентификация:** Регистрация, вход и выход для пользователей.
*   **Просмотр карточек:** Простой режим просмотра с возможностью перевернуть карточку.
*   **Интервальное повторение:** Реализована система Лейтнера для эффективного запоминания.
*   **Режим викторины:** Проверка знаний в формате теста.
*   **Режим письменного ввода:** Проверка правописания ответа.
*   **Статистика:** Отслеживание прогресса обучения с визуализацией.

## Стек технологий

*   **Backend:** Python, Flask
*   **Frontend:** HTML, CSS, Jinja2, JavaScript, Bootstrap 5
*   **База данных:** SQLite (через Flask-SQLAlchemy)
*   **Визуализация:** Chart.js
*   **Безопасность:** Werkzeug Security для хеширования паролей
*   **Контроль версий:** Git, GitHub

## Установка и запуск (локально)

Чтобы запустить проект на своем компьютере, выполните следующие шаги:

1.  **Клонируйте репозиторий:**
    ```
    git clone https://github.com/bbldaaa750/flashcard-app
    cd \flashcard-app
    ```

2.  **Создайте и активируйте виртуальное окружение:**

    *   *Для Windows:*
        ```
        python -m venv venv
        venv\Scripts\activate
        ```
    *   *Для macOS / Linux:*
        ```
        python3 -m venv venv
        source venv/bin/activate
        ```

3.  **Установите зависимости:**
    ```
    pip install -r requirements.txt
    ```

4.  **Запустите приложение:**
    ```
    python run.py
    ```

5.  Откройте браузер и перейдите по адресу `127.0.0.1:5000`.
